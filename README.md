# ⚡ Ohm's Law Calculator

A simple web app that calculates Voltage, Current, or Resistance using Ohm's Law (V = I × R).

Built with HTML, CSS, and JavaScript — no frameworks, no hardware, just code.

## How it works
1. Enter any 2 of the 3 values (Voltage, Current, Resistance)
2. Leave the unknown one empty
3. Click Calculate
